Processo enviar arquivos para o GIT

1) Logar no DOS e posicionar no diretório pm-projetos
C:\Dados\CursoPlayCode-Programação\pm-projetos

2) Criar o envelope dos arquivos para subir.

3) Dar o comando "git init"

4) Dar o comando "git status"

5) Dar o comando "git add . para adicionar os projetos ao envelope

6) Dar o comando "git status" para ver se está tudo adicionado no evelope

7) Dar o comando "git commit -m" "Primeiro commit" para selar o envelope

8) Dar o comando "git log" para ver resultado

9) Digitar o comando "git remote add origin https://github.com/marioeug/pm-projetos.git" (usei este para dar certo: git remote set-url origin https://github.com/marioeug/pm-projetos.git)
para subir para o repositorio no Github)

10) Dar o comando "git push -u origin main" (usei este para dar certo:  git push origin master)
